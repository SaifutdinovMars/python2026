Задание 2. Основы Python. Переменные и типы данных

### Описание
Сделай скрипт, который получает название квеста, требуемое количество очков и фактически набранные очки. Приведи требуемое количество к int, фактическое количество к float. Рассчитай процент выполнения и выведи значения вместе с типами.

### Пример входных данных
```
Training mission
200
145.5
```

### Пример выходных данных
```
Квест: Training mission (str)
Требуется: 200 (int)
Получено: 145.5 (float)
Выполнение: 72.75% (float)
```

### Критерии успешного выполнения
- Три значения читаются через input() в указанном порядке.
- Требуемое количество преобразуется к int, фактическое к float.
- Процент выполнения вычисляется корректно как фактическое деленное на требуемое умноженное на 100.
- Вывод содержит подписи, данные и типы.
- Скрипт завершается без ошибок при корректном вводе и положительном требуемом количестве очков.

### Подсказки
- Используй отдельную переменную для процента, чтобы потом вывести ее тип.
- Применяй f-строки для форматирования.
- Проверяй, что делитель не равен нулю, и при необходимости выводи сообщение и завершай работу.

### Частые ошибки и ограничения
- Не перепутай преобразование int и float.
- Не округляй процент произвольно. оставь стандартное поведение float.
- Не добавляй дополнительные операции кроме требуемых.
