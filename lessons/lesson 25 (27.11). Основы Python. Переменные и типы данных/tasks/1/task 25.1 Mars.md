Задание 1. Основы Python. Переменные и типы данных

### Описание
Напиши программу, которая принимает имя, возраст и уровень энергии игрока. Имя оставь строкой. возраст приведи к int. уровень энергии приведи к float. Выведи каждое значение с подписью и типом.

### Пример входных данных
```
Mars
15
82.5
```

### Пример выходных данных
```
Имя: Mars (str)
Возраст: 15 (int)
Энергия: 82.5 (float)
```

### Критерии успешного выполнения
- Три вызова input() выполняются в указанном порядке.
- Возраст корректно преобразуется к int.
- Уровень энергии преобразуется к float.
- Вывод содержит подписи, значения и типы.
- Скрипт завершается без исключений при корректном вводе.

### Подсказки
- Храни результат type(...)__name__ в переменной, чтобы не повторять код.
- Используй f-строки для читаемого вывода.
- Проверь порядок преобразований, чтобы избежать ValueError.

### Частые ошибки и ограничения
- Не добавляй лишние сообщения в вывод.
- Не пропускай преобразование к числам, иначе тип будет str.
- Не меняй порядок ввода значений.
