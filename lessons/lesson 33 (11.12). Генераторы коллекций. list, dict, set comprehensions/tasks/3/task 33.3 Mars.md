# Задание 33.3. Генераторы коллекций. list, dict, set comprehensions

## Блок урока
«Dict comprehensions» показывает. как из списка словарей быстро собрать новый словарь с готовыми значениями.

## Условие задачи
**Сложность**: <span style="color: #2e7d32">● лёгкий</span>. примерное время: 15–20 минут

Есть список `students`, где каждый элемент — словарь с ключами `"name"` и `"score"`. Построй одним dict comprehension словарь `result`, в котором ключ — имя, а значение — строка `"сдал"`, если балл не ниже 60, иначе `"нужно повторить"`.

## Пример входных данных
```
students = [
    {"name": "Вика", "score": 75},
    {"name": "Петя", "score": 55},
    {"name": "Ира", "score": 90}
]
```

## Пример выходных данных
```
{'Вика': 'сдал', 'Петя': 'нужно повторить', 'Ира': 'сдал'}
```

## Критерии успешного выполнения
- Словарь `result` формируется одним выражением dict comprehension.
- Используй условие внутри выражения, чтобы выбрать нужную строку.
- Исходный список `students` не изменяется.
- В результате есть записи для всех студентов.

## Подсказки
- Пройдись по списку словарей так: `for student in students`.
- Значение выбирай через тернарный оператор `"сдал" if score >= 60 else "нужно повторить"`.
- Не забывай брать имя через `student["name"]` и балл через `student["score"]`.

## Частые ошибки и ограничения
- Не добавляй внешний цикл и `dict()` вокруг comprehension.
- Следи, чтобы ключи словаря не повторялись.
- Не модифицируй исходные словари внутри списка.
