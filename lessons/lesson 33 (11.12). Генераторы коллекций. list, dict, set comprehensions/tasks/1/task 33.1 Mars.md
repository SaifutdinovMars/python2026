# Задание 33.1. Генераторы коллекций. list, dict, set comprehensions

## Блок урока
«Введение в генераторы коллекций» показывает. как короткое выражение заменяет длинный цикл и помогает сконцентрироваться на сути преобразования данных.

## Условие задачи
**Сложность**: <span style="color: #2e7d32">● лёгкий</span>. примерное время: 15–20 минут

Есть список городов с лишними пробелами и случайным регистром. Сформируй новый список `clean_cities`, где каждый город записан без пробелов в начале и конце и с заглавной первой буквой. Используй одно выражение list comprehension.

## Пример входных данных
```
raw_cities = ["  omSk", "PERM  ", " kazan ", "Саратов"]
```

## Пример выходных данных
```
['Omsk', 'Perm', 'Kazan', 'Саратов']
```

## Критерии успешного выполнения
- Преобразование делается только через list comprehension.
- Все пробелы по краям убраны у каждого элемента.
- Первая латинская буква становится заглавной. остальные строчными.
- Русские названия остаются как есть. но без пробелов.

## Подсказки
- Сначала потренируй выражение `city.strip()` для удаления пробелов.
- Затем примени метод `title()` к строке.
- Объедини эти действия внутри comprehension `[...]`.
- Вспомни. что синтаксис выглядит как `[выражение for элемент in список]`.

## Частые ошибки и ограничения
- Не добавляй `append`. результат должен вернуться из comprehension.
- Не меняй исходный список `raw_cities`.
- Следи. чтобы выражение было читаемым: одно преобразование и одна коллекция.
