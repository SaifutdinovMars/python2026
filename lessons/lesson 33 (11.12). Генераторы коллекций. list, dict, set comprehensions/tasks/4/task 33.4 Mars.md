# Задание 33.4. Генераторы коллекций. list, dict, set comprehensions

## Блок урока
«Set comprehensions» показывает. как быстро получить уникальные значения и убрать лишнее без длинного цикла.

## Условие задачи
**Сложность**: <span style="color: #2e7d32">● лёгкий</span>. примерное время: 15–20 минут

Дан список `raw_colors`, где встречаются повторяющиеся строки и лишние пробелы. Построй множество `clean_colors`, в которое попадут только непустые значения в нижнем регистре без пробелов по краям. Используй одно выражение set comprehension.

## Пример входных данных
```
raw_colors = ["  RED", "blue ", "", "Blue", " green", "GREEN "]
```

## Пример выходных данных
```
{'red', 'blue', 'green'}
```

## Критерии успешного выполнения
- Результат создаётся одним set comprehension.
- Пустые строки и строки из пробелов исключаются.
- Все элементы приводятся к нижнему регистру.
- Исходный список `raw_colors` не меняется.

## Подсказки
- Используй `strip()` чтобы убрать пробелы с краёв.
- Проверяй непустоту после `strip()`.
- Применяй `lower()` к уже очищенной строке.

## Частые ошибки и ограничения
- Не оборачивай выражение в `set()` и не добавляй `add`.
- Не забудь, что множеству не важен порядок, поэтому всё делай спокойно.
- Следи, чтобы списки не содержали более пяти пунктов.
