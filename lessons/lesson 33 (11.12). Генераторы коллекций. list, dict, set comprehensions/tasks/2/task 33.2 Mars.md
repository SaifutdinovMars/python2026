# Задание 33.2. Генераторы коллекций. list, dict, set comprehensions

## Блок урока
«List comprehensions» помогает заменить длинные циклы. ты сразу видишь, какие элементы берёшь и как их меняешь.

## Условие задачи
**Сложность**: <span style="color: #2e7d32">● лёгкий</span>. примерное время: 15–20 минут

Есть список `sensor_data` с парами `(номер_датчика, значение)`. Нужно получить список строк `alerts`, где попадут только значения выше 80. Формат каждой строки: `"Датчик <номер>: <значение>"`. Сделай это одним выражением list comprehension.

## Пример входных данных
```
sensor_data = [
    (1, 75),
    (2, 88),
    (3, 95),
    (4, 40)
]
```

## Пример выходных данных
```
['Датчик 2: 88', 'Датчик 3: 95']
```

## Критерии успешного выполнения
- Список `alerts` формируется одним list comprehension.
- В результат попадают только пары с значением > 80.
- Формат строки полностью совпадает с образцом.
- Исходный список `sensor_data` не меняется.

## Подсказки
- Вспомни шаблон `[выражение for пара in список if условие]`.
- Распакуй пары прямо в comprehension: `for sensor_id, value in sensor_data`.
- Фильтр `if value > 80` поставь в конце выражения.
- Используй f-строку `f"Датчик {sensor_id}: {value}"`.
- Сохрани результат в новой переменной, не перезаписывай входные данные.

## Частые ошибки и ограничения
- Нельзя добавлять `append`. весь результат появляется внутри comprehension.
- Не пиши несколько условий подряд, хватит одного сравнения.
- Не меняй порядок исходных элементов.
