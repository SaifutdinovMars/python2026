## Задача 1. Каталог игр

Магазин игр хранит цены в виде списка пар.

Пример входных данных:
```python
games = [
    ("Ведьмак", 1299),
    ("Minecraft", 999),
    ("Stardew", 699),
]
request = ["Minecraft", "Cyberpunk"]
```

### Требуется
1. Создать словарь `prices`, где ключ. название игры, значение. цена.
2. Для каждого названия из списка `request` вывести `<название> -> <цена или "нет в продаже">`.
3. Использовать `get` с запасным значением.

## Задача 2. Баланс по кошелькам

Финдепартамент хранит операции по кошелькам пользователей.

Пример входных данных:
```python
operations = [
    ("user1", +1000),
    ("user2", +500),
    ("user1", -300),
    ("user3", +200),
]
closed = ["user2", "ghost"]
```

### Требуется
1. Собрать словарь `balances`, где ключ. id пользователя, значение. итоговый баланс.
2. Удалить из словаря все id из списка `closed` через `pop(user_id, 0)`.
3. Вывести итоговый словарь.

### Критерии
- Баланс считается суммированием значений по ключу.
- Удаление безопасно для несуществующих id.

## Задача 3. Подписки на рассылку

Есть два множества email‑адресов: активные подписчики и участники последнего вебинара.

```python
subscribers = {"a@example.com", "b@example.com", "c@example.com"}
webinar = {"b@example.com", "d@example.com"}
```

### Требуется
1. Найти пользователей, которые и подписаны, и были на вебинаре.
2. Найти пользователей, которые были только на вебинаре.
3. Распечатать результат с понятными подписями.

### Требуется
