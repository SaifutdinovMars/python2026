### Задание 31.3. Основы Python. Работа со строками и списками

### Описание
Напиши программу, которая упорядочивает список никнеймов игроков.

Требования:
- прочитать строку с никнеймами, разделенными запятой
- удалить пробелы вокруг каждого никнейма
- отфильтровать никнеймы короче 3 символов
- привести все никнеймы к нижнему регистру
- отсортировать список по длине, затем по алфавиту

### Пример входных данных
```
  Max, g,  rocket,  pa, Mars,  pythonista 
```

### Пример выходных данных
```
['max', 'mars', 'rocket', 'pythonista']
```

### Критерии успешного выполнения
- строка корректно разбита по запятой
- пробелы по краям убраны
- короткие никнеймы отфильтрованы
- итоговый список отсортирован по длине и алфавиту

### Подсказки
- используй `split(',')`, `strip()`, `lower()`
- для сортировки можно использовать параметр `key`

### Частые ошибки и ограничения
- не забывай сначала нормализовать строку, а потом сортировать
- не используй внешние библиотеки . хватит стандартного Python
