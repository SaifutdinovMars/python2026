### Задание 31.1. Основы Python. Работа со строками и списками

### Описание
Напиши программу, которая принимает строку с произвольным текстом и приводит ее к аккуратному виду.

Требования:
- удалить пробелы в начале и в конце строки
- заменить несколько пробелов внутри строки на один
- привести весь текст к виду, где первая буква строки заглавная, остальные строчные
- вывести исходную длину строки и длину после нормализации

### Пример входных данных
```
   пИтОн   ЭТо   ЗДОРОВО    
```

### Пример выходных данных
```
Нормализованный текст: Python это здорово
Исходная длина: 24
Новая длина: 21
```

### Критерии успешного выполнения
- строка очищена от лишних пробелов по краям
- последовательности из нескольких пробелов заменены на один
- регистр символов приведен к требуемому виду
- длины посчитаны корректно

### Подсказки
- используй `strip`, `split` и `' '.join(...)`
- для регистра можно комбинировать `lower` и работу с первым символом

### Частые ошибки и ограничения
- не пытайся напрямую менять символы строки по индексу
- учитывай, что строка может быть пустой после очистки
