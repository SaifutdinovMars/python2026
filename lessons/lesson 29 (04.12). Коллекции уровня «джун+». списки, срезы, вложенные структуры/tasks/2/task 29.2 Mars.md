# Задание 29.2. Коллекции уровня «джун+». списки, срезы, вложенные структуры

## Блок урока
Индексация и срезы. Отрабатываем чтение списков по позициям: берем нужные сегменты, используем отрицательные индексы и шаг.

## Условие задачи
У тебя есть расписание поездок в трамвае по минутам прибытия. Нужно:
1. Получить первые три расписания для плаката на остановке.
2. Показать последние четыре времени, чтобы проверить вечерние рейсы.
3. Сформировать срез, который показывает рейсы с индекса 2 по 6 включительно.
4. Создать срез с шагом 2, начиная с первого элемента, чтобы выделить только четные по порядку рейсы.

## Пример входных данных
```python
schedule = ["07:05", "07:23", "07:41", "07:59", "08:17", "08:35", "08:53", "09:11", "09:29"]
```

## Пример выходных данных
```
Утро: ['07:05', '07:23', '07:41']
Вечер: ['08:35', '08:53', '09:11', '09:29']
Дневной отрезок: ['07:41', '07:59', '08:17', '08:35', '08:53']
Четные рейсы: ['07:05', '07:41', '08:17', '08:53']
```

## Критерии успешного выполнения
- Все выборки сделаны через срезы
- Используются как позитивные, так и отрицательные индексы
- Каждый срез подписан отдельной строкой
- Исходный список не изменяется

## Подсказки
- `[:3]` быстро берет начало списка
- Отрицательные индексы помогают достать хвост `[-4:]`
- Чтобы включить элемент с позиции 6, ставь `stop=7`
- Шаг `2` позволяет пропускать элементы

## Частые ошибки и ограничения
- Не перепутай порядок `start` и `stop`
- Помни, что `stop` не входит в результат
- Шаг не может быть нулем
- Не перезаписывай список, только читай его частями
