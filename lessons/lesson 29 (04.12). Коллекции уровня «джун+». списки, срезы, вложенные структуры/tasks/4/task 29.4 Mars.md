# Задание 29.4. Коллекции уровня «джун+». списки, срезы, вложенные структуры

## Блок урока
Вложенные структуры. Учимся хранить связанные данные в списках словарей и извлекать вложенные элементы без путаницы.

## Условие задачи
Ты координируешь тренировочный лагерь киберкоманды:
1. Создай список словарей с ключами `day`, `focus`, `slots` (список временных окон) и `coach`.
2. Выведи расписание строками `День: тема. тренер: имя. слоты: ...`.
3. Собери единый список всех временных окон, проходя по вложенным спискам `slots`.
4. Отфильтруй дни для тренера из переменной `coach_filter`.
5. Подсчитай общее количество слотов и выведи его после цикла.

## Пример входных данных
```python
camp = [
    {"day": "Пн", "focus": "реакция", "slots": ["11:00", "14:00"], "coach": "Ева"},
    {"day": "Ср", "focus": "коммуникация", "slots": ["10:00", "13:00", "16:00"], "coach": "Макс"},
    {"day": "Пт", "focus": "тактика", "slots": ["12:00"], "coach": "Ева"}
]
coach_filter = "Ева"
```

## Пример выходных данных
```
Пн: реакция. тренер: Ева. слоты: 11:00, 14:00
Ср: коммуникация. тренер: Макс. слоты: 10:00, 13:00, 16:00
Пт: тактика. тренер: Ева. слоты: 12:00
Все слоты: ['11:00', '14:00', '10:00', '13:00', '16:00', '12:00']
Дни Ева: [{'day': 'Пн', 'focus': 'реакция', 'slots': ['11:00', '14:00'], 'coach': 'Ева'}, {'day': 'Пт', 'focus': 'тактика', 'slots': ['12:00'], 'coach': 'Ева'}]
Всего слотов: 6
```

## Критерии успешного выполнения
- Расписание хранится в списке словарей с одинаковыми ключами
- Вывод расписания реализован циклом со вложенным перебором
- Фильтр по тренеру строится без изменения исходного списка
- Подсчет слотов выполняется программно

## Подсказки
- Для общего списка слотов используй `all_slots.extend(entry["slots"])`
- Проверяй тренера условием `if entry["coach"] == coach_filter`
- Счетчик можно увеличивать на `len(entry["slots"])`

## Частые ошибки и ограничения
- Не обращайся к несуществующим ключам в словаре
- Не перезаписывай исходный список при фильтрации
- Не модифицируй вложенные списки, если они еще нужны
- Не копируй код для каждого дня. достаточно циклов
