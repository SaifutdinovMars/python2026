# Задание 29.1. Коллекции уровня «джун+». списки, срезы, вложенные структуры

## Блок урока
Знакомство со списками. Сфокусируйся на создании списка, обращении по индексу и переборе элементов без ошибок выхода за границы.

## Условие задачи
Готовишься к вечеру настольных игр. Создай список игр, которые точно принесешь. Добавь предложения друзей в конец списка. Выведи количество игр, покажи первую и среднюю игру, затем распечатай весь список с номерами, чтобы понимать порядок.

## Пример входных данных
```python
default_games = ["Каркассон", "Ticket to Ride", "Диксит"]
friend_games = ["Экивоки", "Имаджинариум"]
```

## Пример выходных данных
```
Всего игр: 5
Первая игра: Каркассон
Средняя игра: Диксит
1. Каркассон
2. Ticket to Ride
3. Диксит
4. Экивоки
5. Имаджинариум
```

## Критерии успешного выполнения
- Список создается из исходных данных и расширяется методами списка
- Количество элементов выводится через `len`
- Индексация используется для доступа к первому и среднему элементам
- Перебор выводит элементы в отдельной строке с порядковым номером

## Подсказки
- Сохрани средний индекс как `len(list) // 2`, чтобы не считать вручную
- Для расширения списка сразу несколькими играми подойдет `extend`
- Используй `enumerate` для нумерации, чтобы не заводить отдельный счетчик

## Частые ошибки и ограничения
- Не забывай, что индекс `len(list)//2` работает только при непустом списке
- Избегай ручного конкатенирования строк вместо метода добавления
- Не меняй список во время перебора, сначала сформируй полный набор
- Следи, чтобы в выводе не было дополнительных пробелов или символов
